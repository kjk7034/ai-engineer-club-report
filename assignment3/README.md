# 개인화 영화 추천 챗봇

사용자의 취향을 **기억**하는 영화 추천 챗봇입니다. 좋아하는 장르와 이미 본 영화를 기억하고, 대화 기록을 바탕으로 개인화된 추천을 제공합니다.

## 과제 설명

OpenAI 클라이언트와 대화 기록(`messages` 리스트)을 활용해 메모리를 가진 챗봇을 구축합니다.

- **모델**: `gpt-4o-mini`

챗봇이 갖춰야 할 기능:

- 사용자가 좋아하는 장르를 기억
- 이미 시청한 영화를 기억
- 대화 기록을 기반으로 개인화된 추천 제공 (이미 본 영화는 추천하지 않음)

### 동작 방식

1. `messages` 리스트가 챗봇의 메모리 역할을 합니다. 첫 메시지로 추천 규칙을 담은 시스템 프롬프트를 넣습니다.
2. 사용자 메시지(`role: "user"`)와 AI 응답(`role: "assistant"`)을 모두 `messages`에 append합니다.
3. 매 호출마다 **대화 기록 전체**를 모델에 전달하므로, 모델이 앞선 맥락(장르 취향, 시청한 영화)을 그대로 기억합니다.
4. 시스템 프롬프트에서 "이미 본 영화는 절대 추천하지 않는다"를 규칙으로 강제합니다.

## 대화 예시

```
User: 나는 SF 영화를 좋아해
AI: 좋은 취향이시네요! SF에는 명작이 정말 많죠...

User: 인셉션이랑 인터스텔라는 이미 봤어
AI: 좋은 선택이셨네요! 이미 보셨으니까...

User: 오늘 밤에 뭐 볼지 추천해 줄래?
AI: SF를 좋아하시고, 인셉션과 인터스텔라는 이미 보셨으니까 추천드리자면...

User: 내가 좋아하는 장르랑 이미 본 영화가 뭐라고 했지?
AI: SF를 좋아하신다고 했죠! 인셉션과 인터스텔라를 보셨습니다.
```

마지막 셀에서 `messages` 리스트를 순회하면 사용자와 AI의 대화가 모두 누적되어 있는 것을 확인할 수 있습니다.

## 파일 구조

- `pyproject.toml` — 프로젝트 메타데이터 및 의존성
